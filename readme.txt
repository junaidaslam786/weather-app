1. first create a virtual environment by using this command
    py -m venv .venv

2.  now activate the venv by using this command 
  .venv/Scripts/activate

3. then you have to create an account on this website for getting the weather information
   using api key
    https://openweathermap.org/

4. you should have to create .env file in your project where you can specify your secrets keys
    API_KEY_FOR_WEATHER='9e1defe6e8e6b6e577dbe8adae6f751c'
